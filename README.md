# Lakshi Online Book Shop

## Project Description
A dynamic book shop application built with PHP, MySQL, HTML, CSS, and Tailwind CSS. Features include user registration, secure login, and a contact messaging system.

## Setup Instructions

1. **Prerequisites**: Ensure you have XAMPP, WAMP, or any other local server installed with PHP and MySQL.
2. **Clone/Download Project**: Place the project folder into your server's root directory (`htdocs` for XAMPP or `www` for WAMP).
3. **Database Import**:
    - Open `phpMyAdmin` (typically at `http://localhost/phpmyadmin`).
    - We have provided the SQL initialization file in `database.sql`. You can simply import it in the 'Import' tab or copy it into the SQL tab. It will automatically create the `lakshi_bookshop` database alongside necessary tables (`users`, `products`, `contact_messages`) and insert sample products.
4. **Run Application**:
    - Open your browser.
    - Navigate to `http://localhost/Lakshi_Online_Book_Shop` (or your specific root folder name if you renamed it).
